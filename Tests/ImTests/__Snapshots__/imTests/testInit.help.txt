USAGE: im [--toggle] [--list] [--last] [--list-id] [--list-alfred] [<id>] [--debug <debug>] [--set-last <set-last>]

ARGUMENTS:
  <id>                    The ID to select

OPTIONS:
  --toggle                toggle last input method by im
  --list                  show list
  --last                  show last id for toggle
  --list-id               show id list
  --list-alfred           show list as Alfred Format
  --debug <debug>         to inject test env
  -s, --set-last <set-last>
                          Set the name to toggle
  --version               Show the version.
  -h, --help              Show help information.
